## 📌 Sobre el Proyecto

**España Viva** es un dashboard web interactivo diseñado para mostrar, con datos empíricos, los progresos y logros extraordinarios de España en índices clave de calidad de vida, sostenibilidad, salud pública y convivencia, en comparación con los países de la OCDE.

Frente al ruido de las noticias negativas, esta aplicación busca ofrecer una visión más amplia, equilibrada y basada en datos verificados de instituciones como el Banco Mundial, la OCDE y el INE.

## 🚀 Características y Funcionalidades Principales

- **Fichas de Bienestar Interactivas:** Visualizaciones inmediatas de hitos importantes de tolerancia, salud y educación, presentados con tarjetas con animación *flip*.
- **Radar Global (Comparativa OCDE):** Gráficos SVG interactivos construidos con Recharts que desglosan la esperanza de vida, seguridad ciudadana, inversión renovable y cohesión social frente a la media de la OCDE.
- **Micro-Animaciones e Interfaces Zen:** Transiciones suaves usando Framer Motion, logrando un diseño sereno y relajante ("Modo Noche Zen" incluido).
- **Gamificación Sutil:** Logros desbloqueables en función de las métricas exploradas ("Buscador de la Convivencia", "Analista del Sistema", etc.).
- **Mapa Interactivo de España:** Creado con D3 y react-simple-maps para mostrar disparidades y datos a nivel autonómico, enriquecidos de un profundo análisis de finanzas territoriales.
- **Analogía Financiera del Estado:** "El Estado como un Hogar", simplificando los presupuestos generales del Estado a escala ciudadana.

## 🛠️ Stack Tecnológico

La aplicación está construida aplicando modernos principios de desarrollo Front-End:
- **Core:** React 19 + TypeScript
- **Bundler:** Vite
- **Estilos:** Tailwind CSS v4 con variables CSS personalizadas
- **Componentes Visuales:** Lucide React (Íconos), Framer Motion (Animaciones de sistema)
- **Visualización de Datos:** Recharts, D3-scale, React Simple Maps

## ⚙️ Cómo ejecutar en local

```bash
# 1. Clona el repositorio
git clone <url-del-repositorio>
cd data-happiness

# 2. Instala las dependencias
npm install
# o
yarn

# 3. Arranca el servidor de desarrollo
npm run dev
# o
yarn dev
```

El servidor local se ejecutará en http://localhost:5173 por defecto.
